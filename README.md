# Fashion Sales Forecasting

This project predicts future fashion product sales using time series forecasting.

## Tools Used
- Python
- Pandas
- Prophet
- Matplotlib
- Jupyter Notebook

## Files
- Fashion_forecasting.ipynb : Main notebook
- sales_data.xlsx : Sales dataset
- forecast.png : Forecast output image

## Objective
To analyze past sales data and forecast future demand to support business decisions.
